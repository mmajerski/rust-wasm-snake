## Simple Snake Game

### Description

This implementation of popular snake game was done while exploring topics related to WebAssembly with Rust as a backend language.

### Main tools used

- WebAssembly
- Rust
- TypeScript
- Webpack
- Frontend is rendered by CanvasAPI
